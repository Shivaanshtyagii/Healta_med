# Healta - Medication Management System

Healta is a modern, full-stack web application designed to help users track their daily medication schedules, manage stock levels, and organize refills. It features a clean, Apple-inspired minimalist UI with smooth animations and real-time notifications.

## üöÄ Features

* **Daily Schedule Dashboard**: View and manage medications scheduled for today.
* **Inventory & Stock Tracking**: Keep track of medication quantities and refill requirements.
* **Add/Edit Medications**: Simplified forms to add new prescriptions or update existing ones.
* **Real-time Notifications**: Visual feedback for user actions via toast notifications.
* **Animated UI**: Fluid page transitions and interactive elements powered by Framer Motion.
